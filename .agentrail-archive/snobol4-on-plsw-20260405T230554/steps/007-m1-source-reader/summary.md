Implemented SNOBOL4 source reader and lexer in PL/SW. Source reader (SRC_INIT/PEEK/NEXT) with position-based traversal. Lexer (LEX_NEXT) tokenizes: identifiers (auto-uppercased), integer literals, string literals (single-quoted), operators (= + - , : ( )), EOL, EOF. Character classification helpers (IS_ALPHA/DIGIT/ALNUM). Token name lookup for display. Tested on 4-line SNOBOL4 program producing correct 17-token stream. Filed plsw#18 (source buffer, fixed).
