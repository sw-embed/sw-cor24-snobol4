Implemented user-defined functions end-to-end in modular interpreter: DEFINE('FN(P)') registers in fn table, FN(arg) call expression lowers to OP_CALL, :(RETURN)/:(FRETURN) lower to OP_RET/OP_FRET, call frame stack saves PC + param across nested calls, return value comes from function-name var per SNOBOL4 convention. examples/factorial.sno (DBL doubler) prints 42. Limits: 1 arg, 4 fns, depth 16. All prior examples still pass. Recursion works in principle but needs conditional builtins (step 2) for factorial-style demos.
